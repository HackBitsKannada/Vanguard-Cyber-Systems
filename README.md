A high-end, responsive, single-page website built for a premium cybersecurity firm. This project focuses on modern UI/UX principles, featuring a "dark-mode" aesthetic and interactive elements to build user trust and engagement.
🚀 Key Features
Modern UI/UX Design: Utilizes a "Dark Mode" theme with glassmorphic navigation and electric cyan accents for a high-tech, professional feel.
Mobile-First Responsiveness: Fully optimized for all screen sizes, including a dedicated hamburger menu for mobile navigation.
Interactive Animations: Features scroll-reveal animations that trigger as the user explores the page, alongside a custom-designed cursor for desktop users.
Global Network Dashboard: Includes a live-style monitoring panel showing system health, threat levels, and uptime metrics.
Service Grid: A clean, 3-column layout highlighting core business offerings: Vulnerability Assessments, Network Hardening, and Incident Response.
🛠️ Tech Stack
HTML5: Semantic structure for better SEO and accessibility.
CSS3: Custom properties (variables), Flexbox, and CSS Grid for a robust, modern layout.
JavaScript (Vanilla): Lightweight scripts for the Intersection Observer (scroll reveal), custom cursor tracking, and mobile menu toggling.
Typography: Premium look using JetBrains Mono for technical data and Inter for clean readability.
📂 Project Structure
vanguard-cyber-systems.html: The core file containing all HTML structure, CSS styling, and JavaScript logic.
Assets: Uses embedded SVG icons to ensure fast loading times and crisp visuals on high-resolution displays.
💻 How to View
Download the vanguard-cyber-systems.html file.
Open the file in any modern web browser (Chrome, Firefox, Safari, or Edge).
Scroll down to experience the reveal animations and interactive dashboard.
